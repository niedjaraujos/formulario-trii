# 📄 Formulário de Contato

Projeto desenvolvido durante estudos em desenvolvimento front-end, com foco em validação de formulários.

---

## 📌 Contexto

Este projeto foi iniciado como exercício prático de curso e evoluído para consolidar conceitos importantes de:

* HTML semântico
* CSS e responsividade
* JavaScript para validação de formulário

---

## 🚀 Funcionalidades

* Estrutura semântica com HTML5
* Formulário com campos obrigatórios
* Associação correta entre `label` e `input`
* Validação de campos com JavaScript
* Feedback ao usuário ao enviar o formulário

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (Flexbox + responsividade)
* JavaScript (manipulação do DOM)

---

## 💡 Aprendizados

* Importância da semântica no HTML
* Estrutura correta de formulários acessíveis
* Manipulação de eventos com JavaScript
* Separação de responsabilidades (HTML, CSS e JS)
* Boas práticas para validação de formulários
